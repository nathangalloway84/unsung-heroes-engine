# The Unsung Heroes Engine
**Team USA x Google Cloud Hackathon Submission**

The Unsung Heroes Engine is an interactive, dynamic dashboard that highlights the "Hidden Grind" of non-mainstream Olympic and Paralympic athletes. It uses the Google Cloud Gemini API to generate inspiring "Journey Archetypes" based on historical, public Team USA data without violating Name, Image, and Likeness (NIL) bans.
